# Standup game

This is a simple game template, Phaser framework is already implemented.

## Game Mechanics

- a stack game
- the player should stop the rectangle on top of each other
- if it's not perfectly on top it will cut a part of it
- click or press space bar to stack
- when the player get it wrong the game ends
- the speed will increase every 5 blocks 
- has a tutorial so the player will know how to play 
- player can try again
